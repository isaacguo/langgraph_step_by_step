# Project 5: Error Handling & Failure Recovery

## Overview
Learn robust error handling, retry mechanisms, and failure recovery strategies.

## Concepts Covered
- Error handling patterns
- Retry mechanisms
- Circuit breakers
- Fallback strategies
- Error propagation
- Graceful degradation
- Recovery workflows

## Files
- `01_basic_error_handling.py` - Basic error handling
- `02_retry_mechanisms.py` - Retry patterns
- `03_circuit_breaker.py` - Circuit breaker pattern
- `04_fallback_strategies.py` - Fallback mechanisms
- `05_recovery_workflows.py` - Recovery workflows

## Running Examples

```bash
conda activate langgraph_exec
cd project_5_error_handling
python 01_basic_error_handling.py
```

## Prerequisites
Complete Projects 1-4 before starting this project.
